toString()

If we pass an array in println function in java, it diplays something that is not human readable.

toString() method of arrays class can convert an arrya into string which is in human readable form.

Syntax Arrays.toString(array_variable)

sort()

It sorts the given array.

Syntax Arrays.sort(array_variable)

Note: It doesn't return anything, just sorts the original array.

binarySearch():

It returns you the index of the given parameter

Syntax Arrays.binarySearch(array_variable, element)

Note: the array passed must be sorted.
