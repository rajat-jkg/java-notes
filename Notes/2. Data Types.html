Data Types

Primitive Data Types

Premitive data types are the predefined data types in. There are following predefined data types in java.

Data Type Size Description
byte 1 byte Stores whole numbers from -128 to 127
short 2 bytes Stores whole numbers from -32,768 to 32,767
int 4 bytes Stores whole numbers from -2,147,483,648 to 2,147,483,647
long 8 bytes Stores whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
float 4 bytes Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits
double 8 bytes Stores fractional numbers. Sufficient for storing 15 decimal digits
boolean 1 bit Stores true or false values
char 2 bytes Stores a single character/letter or ASCII values

Java is called a type safe language because every variable in Java has a fixed data type.

Using float and long data tpes

Long variable is followed by l and float is followed by f

Example:

float f = 1.2f;
long l = 2222222l;

char data type

char is enclosed by single quotes. If we assign an integer to a char variable it stores the character corresponding to that ASCII(American Standard Code for information exchange) code.

Type casting

It is the conversion of one data type to another.
Syntax:

(data_type)value

Here the povided value gets converted into the data type given in parenthesis.
